# Laboratorio 5 - Estructuras de Datos Dinámicas

Este repositorio contiene la implementación del Laboratorio 5 del curso IE-0117 Programación Bajo Plataformas Abiertas.

## Contenido

El proyecto implementa las siguientes estructuras de datos en C:

- Arreglos dinámicos
- Listas enlazadas simples
- Listas doblemente enlazadas
- Stack

## Estructura del repositorio

```text
include/   Archivos de encabezado .h
src/       Archivos fuente .c
Makefile   Archivo para compilar el proyecto
README.md  Descripción del proyecto
```

## Compilación

```bash
make
```

## Ejecución

```bash
./main
```

## Limpieza

```bash
make clean
```

## Revisión con Valgrind

```bash
make valgrind
```
